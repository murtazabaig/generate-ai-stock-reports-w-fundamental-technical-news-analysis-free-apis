# Generate AI Stock Reports w/ Fundamental, Technical, & News Analysis (Free APIs)

Advanced n8n automation for Generate AI Stock Reports w/ Fundamental, Technical, & News Analysis (Free APIs).

## Overview
- Category: Crypto Trading, AI RAG
- Complexity: advanced
- Source: n8n workflow template export

## What This Automation Does
Automate advanced stock analysis with AI: combine technical, fundamental, and news sentiment data into actionable reports delivered to your inbox. Learn more!

## Included Files
- `workflow.json`

## Setup
1. Import `workflow.json` into n8n.
2. Configure required credentials for the services used in the workflow nodes.
3. Update any environment variables or static values inside nodes (API keys, URLs, IDs).
4. Run a test execution and then activate the workflow.
